# Basic Usage Sample

Drop `BasicHapticsDemo` onto any empty GameObject in a scene and press Play.

| Key | Event |
|-----|-------|
| 1 | Click |
| 2 | Success |
| 3 | Failure |
| 4 | Achievement |
| 5 | ImpactMedium |

If the companion plugin is not installed the console will print `Available: False` — the game keeps running and no haptic fires.
