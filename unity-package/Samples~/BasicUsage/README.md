# Basic Usage Sample

`BasicHapticsDemo`'yu bir sahnedeki boş GameObject'e ekle, Play'e bas.

| Tuş | Event |
|-----|-------|
| 1 | Click |
| 2 | Success |
| 3 | Failure |
| 4 | Achievement |
| 5 | ImpactMedium |

Plugin yüklü değilse console'da `Available: False` yazar; oyun çökmez, haptic tetiklenmez.
